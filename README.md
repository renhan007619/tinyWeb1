# tinyWeb
一个基于 Vue.js 构建的个人技术博客，用于分享开发，技术实践与学习心得。

## 项目功能

### 前端功能
- **页面初始化**：单页应用结构、响应式布局
- **UI 美化**：主题切换（明暗模式）、动画效果、卡片式设计
- **访问统计**：实时访客统计、独立访客数、最后访问时间
- **用户认证**：注册/登录/登出、JWT Token 认证
- **备忘录**：增删改查、状态标记
- **留言板**：留言发布、历史留言展示
- **专注计时器**：番茄钟计时、标签分类、数据统计、历史记录、桌面通知
- **Flappy Bird 游戏**：Canvas 游戏、计分系统

### 后端功能
- **RESTful API**：6 个模块（visit/auth/todo/guestbook/setting/focus）
- **数据库**：MySQL + GORM、连接池管理
- **认证中间件**：JWT 验证、权限控制
- **密码安全**：bcrypt 加密

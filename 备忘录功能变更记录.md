# 备忘录功能变更记录

> 创建日期：2026-04-02  
> 修改文件：`index.html`

---

## 功能概述

在页面右侧栏的「时间胶囊」卡片内，新增一个**备忘录**功能：
- 时间胶囊右上角新增一个 📋 图标按钮，带红色角标显示未完成事项数量
- 点击图标，面板从**时间胶囊左侧**弹出（不占用时间胶囊内部空间）
- 支持「生活」「学习」「*** 重要」三个分类标签页切换
- 「*** 重要」用于记录长期重要任务
- 支持添加、完成勾选、双击编辑、删除事项
- 数据自动保存到浏览器 `localStorage`，刷新页面不丢失
- 每天自动归档当天任务到「归档箱」，可翻看历史日记

### 关键设计点

| 样式 | 用途 | 学习要点 |
|------|------|----------|
| `position: absolute; right: calc(100% - 1rem)` | 面板定位到时间胶囊左侧 | `calc()` 函数计算位置，`100%` 指父元素宽度 |
| `transform: translateX(12px) scale(0.96)` | 默认偏移+缩小状态 | CSS transform 可以同时应用多个变换 |
| `pointer-events: none` | 面板收起时不可点击 | 防止透明面板挡住其他元素 |
| `backdrop-filter: blur(14px)` | 毛玻璃效果 | CSS 滤镜，需要配合半透明背景 |
| `z-index: 19` / `z-index: 21` | 层级控制 | 历史面板在备忘录面板之上 |

## 第二次变更（v2）：面板移到左侧 + 新增重要标签 + 历史归档

### 变更一：面板从内部展开改为左侧弹出

**原设计**：面板在时间胶囊卡片内部，向下展开（`max-height` 动画）
**新设计**：面板作为绝对定位元素，从时间胶囊**左侧**弹出

**关键 CSS 变化**：
```css
/* 原代码 - 面板在卡片内部 */
.todo-panel {
    max-height: 0;
    padding: 0 0;
}
.todo-panel.open {
    max-height: 420px;
    padding: 0.8rem 0 0 0;
}

/* 新代码 - 面板从左侧弹出 */
.todo-panel {
    position: absolute;
    top: -0.5rem;
    right: calc(100% - 1rem);  /* 弹出到左侧 */
    width: 280px;
    max-height: 0;
    transform: translateX(12px) scale(0.96);
    background: var(--card-bg);
    backdrop-filter: blur(14px);
    box-shadow: 0 8px 32px rgba(0,0,0,0.25);
}
.todo-panel.open {
    max-height: 520px;
    transform: translateX(0) scale(1);
    padding: 0.8rem;
}
```

**为什么这样改**：时间胶囊内部空间有限，任务多了需要滚动条。弹出面板不受卡片宽度限制，空间更大。

## 总结

| 版本 | 变更内容 | 技术要点 |
|------|----------|----------|
| v1 | 基础备忘录（生活/学习分类，CRUD，localStorage） | IIFE、DOM 操作、事件委托、CSS 动画 |
| v2 | 面板改为左侧弹出 | `position: absolute` + `calc()` + `transform` |
| v2 | 新增「*** 重要」标签 | 数据结构扩展，标签切换逻辑 |
| v2 | 历史归档功能 | 定时检查跨天、日期计算、归档数据读写 |

**涉及的所有 localStorage 键**：

| 键名 | 用途 | 数据格式 |
|------|------|----------|
| `tinyWeb1_todos` | 当前任务 | `{ life: [], study: [], important: [] }` |
| `tinyWeb1_history` | 历史归档 | `{ "YYYY-MM-DD": { life: [], study: [], important: [] } }` |
| `tinyWeb1_archiveDate` | 上次归档日期 | `"YYYY-MM-DD"` 字符串 |

**如果你想清除所有备忘录数据**，在浏览器控制台执行：
```javascript
localStorage.removeItem('tinyWeb1_todos');
localStorage.removeItem('tinyWeb1_history');
localStorage.removeItem('tinyWeb1_archiveDate');
```
然后刷新页面即可。
